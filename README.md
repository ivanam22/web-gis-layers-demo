# Web GIS Layers Visualization

This project demonstrates a simple Web GIS application built using Leaflet and GeoServer.

## Project Overview
The application explains the difference between raster and vector data in GIS and provides interactive visualization of both layer types.

## Features
- Display of raster layer using WMS service
- Display of vector layer using WMS service
- Interactive map visualization with Leaflet
- Educational explanation of raster and vector GIS data

## Technologies
- HTML
- JavaScript
- Leaflet
- GeoServer (WMS)
- Web GIS

## Purpose
The project was developed as part of a university course focused on GIS application development.
